# Hotel Booking System

## Technology Stack

* React JS
* Spring Boot
* MySQL
* Bootstrap
* Spring Data JPA
* REST APIs

## Features

* Add and View Rooms
* Book Rooms
* Prevent Double Booking
* View Booking History

## Frontend Setup

1. Clone repository
2. Run: npm install
3. Run: npm run dev

## Backend Setup

1. Configure MySQL in application.properties
2. Run: mvn spring-boot:run

## Assumptions

* Single hotel management system
* Users can book only available rooms
* Room availability is validated before booking
* MySQL database is running locally

## AI Tools Used

ChatGPT was used for guidance on project structure, React components, Spring Boot APIs, debugging, and best practices.

## Challenges Faced

* React and Spring Boot integration
* Preventing double booking using date validation logic
* API testing and debugging
